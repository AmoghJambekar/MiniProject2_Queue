# Mini Project 2

## Build Instructions
```bash
g++ -o queue src/main.cpp src/Queue.cpp -I include
```

## Run
```bash
./queue
```

## File Structure
```
MiniProject4_Queue/
├── include/
│   └── Queue.h
├── src/
│   ├── Queue.cpp
│   └── main.cpp
└── README.md
```

## Usage
The program has a menu with the following options:
1. Enqueue - Add element
2. Dequeue - Remove element
3. Peek - View front element
4. Size - Show queue status
5. Print queue
6. Exit
